# Contributing

The official registry is a curated index of verified OrqaStudio plugins. To submit a plugin for inclusion, open a pull request against this repository following the format in the registry's README.

**Contributing to the core project?** All OrqaStudio core development goes through the [orqastudio-dev](https://github.com/orqastudio/orqastudio-dev) repository.

See [CONTRIBUTING.md](https://github.com/orqastudio/orqastudio-dev/blob/main/CONTRIBUTING.md) for setup, workflow, and guidelines.
